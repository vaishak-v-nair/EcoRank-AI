# EcoRank-AI

Monorepo-ready scaffold for the EcoRank-AI project.

## Structure

- `backend/` - backend services, AI pipelines, generators, and configuration.
- `frontend/` - Vite + React + TypeScript frontend scaffold.
- `docs/` - prompts, diagrams, and screenshot placeholders.

## Quick start

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Repository root (optional monorepo scripts)

```bash
npm install
npm run dev:frontend
```
